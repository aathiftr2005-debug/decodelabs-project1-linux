# Project 1: Linux & Command Line Basics

## Overview
Mastered Linux CLI fundamentals required in DevOps environments.

## Commands Practiced
| Command | Purpose |
|---------|---------|
| pwd | Print working directory |
| ls -la | List files with details |
| mkdir -p | Create nested directories |
| cd | Change directory |
| cat | View file contents |
| rm -rf | Remove files/folders |
| chmod | Change file permissions |
| touch | Create empty files |

## Tasks Completed
- Created directory structure /app/logs
- Configured file permissions using chmod
- Practiced file operations (cp, mv, rm)
- Understood Linux FHS (Filesystem Hierarchy)

## Key Learning
CLI is faster and scriptable compared to GUI.
